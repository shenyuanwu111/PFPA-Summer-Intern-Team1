# Week3 Haixin Tan Package Contents

This package contains the Week 3 automated data downloading deliverable.

## Main Deliverable

- `src/credit_data_pipeline/`: reusable Python package for downloading and matching market data, SEC financials, and FRED rates.
- `finance_credit_pipeline/theory.md`: KMV/TTC theory notes and data requirements.
- `README.md`: setup and run instructions.
- `tests/test_credit_data_pipeline.py`: regression tests for SEC extraction and trading-date matching.
- `sample_outputs/`: AAPL validation outputs generated by the main pipeline.

## Reference Material Added From GitHub

- `reference/Week3_Shenyuan_Wu/`: source folder downloaded from:
  `https://github.com/shenyuanwu111/PFPA-Summer-Intern-Team1/tree/main/Week3_Automated_Data_downloading/Week3_Shenyuan_Wu`

The reference folder is kept separate so the final package preserves the GitHub material without mixing its standalone script with the enhanced `credit_data_pipeline` entry point.

## Main Run Command

```bash
PYTHONPATH=src python -m credit_data_pipeline.cli \
  --company AAPL \
  --start 2024-01-01 \
  --end 2024-12-31 \
  --out outputs/credit_data \
  --user-agent "your-name your-email@example.com"
```
